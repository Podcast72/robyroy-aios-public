# Limitations

This repository is intentionally limited.
Those limits are part of its public honesty.

## What is not published here

- the full private RobyRoy AIOS operational codebase
- internal bridge flows
- internal memory stores
- private prompts and operational orchestration material
- local runtime paths and private environment configuration
- internal compat or legacy implementations that are not needed for the public explanation

## Evidence limits

The public examples are curated.
The public tests are adapted.
The public documents are reference-oriented.

That means the repository should not be read as a claim that every internal behavior is fully exposed or that every operational detail is represented one-to-one.

## Coverage limits

Not every flow is covered here in the same depth.
The proof set focuses on:

- official backbone clarity
- pre-tool runtime mediation
- post-tool result handling
- governance and runtime separation
- auditability as a reviewable property

## Claim limits

This repository avoids absolute claims on purpose.
It documents a direction, a public perimeter, and a set of technical proofs.
It should not be interpreted as a statement that every private subsystem is published, stable, or formally complete in this repository.
